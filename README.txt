图像标题生成器 - 使用说明
========================

这是一个用于自动生成亚马逊产品图像标题的工具，包含以下功能：
1. 自动生成图像标题（支持中英文）
2. 与手机型号数据合并
3. 应用上架模板格式
4. 按品牌整理数据

使用方法：
---------

1. 确保您的计算机上安装了Microsoft Excel。
2. 双击运行 "ImageTitleGenerator.exe" 程序。
3. 程序会自动处理以下步骤：
   - 读取配置文件 (config.txt)
   - 处理图像文件夹中的图片
   - 生成标题并与型号数据合并
   - 应用模板并整理最终结果
4. 处理完成后，结果将保存在 "Result" 文件夹中。

文件夹结构说明：
---------------

- Image Title Generation: 包含所有处理脚本
- Result: 输出文件夹（程序会自动创建）
- Failure: 处理失败的图片记录（程序会自动创建）
- 需要的excel文件: 包含型号.xlsx和Calanthe上架模板.xlsm
- dist: 包含生成的可执行文件
- build: PyInstaller构建过程中使用的临时文件夹

注意事项：
---------

1. 请勿删除或移动程序运行所需的任何文件和文件夹。
2. 如需修改配置，请编辑 "config.txt" 文件。
3. 如需修改提示词，请编辑 "prompt.txt" 文件。
4. 程序运行期间请勿关闭命令行窗口。
5. 如果遇到问题，请检查 "Failure" 文件夹中的记录文件。
